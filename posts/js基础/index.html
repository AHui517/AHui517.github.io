1.7 数组及数组常见方法

数组是对象数据类型，特殊的对象

let array=[12,28,38,47,];
console.dir(array);
/*
 * arr={
 *	  0:12,
 *    1:28,
 *	  2:38,
 *	  3:47,
 *	  length:4
 */

数组实现增删改查

  1. push() 在数组的末尾进行添加，可以是多个；返回值是添加完元素后的长度。pop() 删除最后一项，并返回删除的那一项。

  2. unshift() 在数组的头部进行添加，可以是多个；返回值是添加完元素后的长度。shift() 删除第一项，并返回删除的那一项。

  3. splice() 实现数组的增删改：

    splice(m,n)：从索引m开始删除n个元素

    splice(n) 如果第二个参数n没写，就从索引m删除到最后

    splice(m,n,x) 删除索引m开始，n个元素，并将x放入删除元素的位置

    返回被删除元素组成的数组

数组拼接和查询

不会改变原数组

  1. slice(m,n) 查询索引m到n的值(不包括n)，并返回查询到的以数组返回，只写一个参数则是查询索引m到结尾处。

  2. concat(任意类型) 返回拼接后的新数组，不改变原数组

数组转化为字符串

  1. toString()

  2. join() ,该方法可以指定一个分隔符，例如join(‘+’)

    eval可以把字符串变为js表达式来执行

数组的一些其他方法

  1. reverse() 反转原数组

  2. sort() 排序数组 改变原数组，按照每个元素的第一位进行排序

    sort((a,b)=>a-b) :从小到大排序

  3. 遍历数组

  • 普通for循环
let arr=[2,3]
for(let i=0;i<arr.length;i++){
    console.log(arr[i]);
}
  • foreach
arr.forEach((item,index)=>{
    console.log("索引："+index,"内容"+item);
})

字符串操作

    字符串在js中是值，而不是一个对象
  1. substr(n,m) 从索引n开始截取m个字符，m不写截取到末尾（后面方法也是） 2. substring(n,m) 从索引n开始找到索引为m处(不含m) 3. slice(n,m) 和substring一样，但是这个支持负数索引，负数索引就加上字符串长度 4. toUpperCase()字符串转大写 5. toLowerCase()字符串转小写l 6. split([分隔符]) 把字符串按指定分隔符拆分成数组(和数组中的join对应) 7. replace(new,old) 实现字符串的替换(只替换一个)

1.8 js中的DOM及其基本操作

document object model 文档对象模型

  • document.getElementById() 在文档中获取指定id元素对象
  • [context].getElementsByTagName() 在指定容器中，通过标签获取一组元素集合
  • [context].getElementsByClassName() 在指定容器中，通过样式名获取一组元素的集合
  • document.getElementsByName() 在文档中，通过标签name属性获取一组节点集合。
  • document.head/body/documentElement获取页面的头部/主体/全部
  • [context].querySelector([selector]) 在指定容器中，通过选择器获取指定元素对象
  • [context].querySelectorAll([selector]) 在指定容器中，通过选择器获取指定元素对象集合

js中节点

节点：Node

节点集合：NodeList（getElementsByName/querySelectorAll都是获取节点集合）

    ```页面中所有东西都是节点```

    **常见节点类型**

  • 元素节点

    • nodeType:1
    • nodeName:大写标签名
    • nodeValue:null

  • 文本节点

    • nodeType:3
    • nodeName:'#text’
    • nodeValue:文本内容

  • 注释节点

    • nodeType:8
    • nodeName:'#comment’
    • nodeValue:注释类容

  • 文档节点

    • nodeType:9
    • nodeName:'#document’
    • nodeValue:null

    描述节点之间关系的属性

    • childNodes:获取所有子节点
    • chidren：获取所用元素子节点（子元素标签集合）
    • firstChild：获取第一个子节点
    • lastChild：获取最后一个子节点
    • firstElementChild/lastElementChild：获取第一个/最后一个元素子节点
    • previousSibling：获取上一个节点
    • nextSibling：获取下一个节点
    • parent：获取父节点
    • previousElementSibling/nextElementSibling:获取上一个/下一个元素子节点
